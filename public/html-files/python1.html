Introduction to Python

Python is a high-level, interpreted programming language known for its readability and versatility. Created by Guido van Rossum in 1991, Python has grown to become one of the world's most popular programming languages.

Why Learn Python?

  • Simple, readable syntax that's easy to learn
  • Versatile - used in web development, data science, AI, automation, and more
  • Large supportive community and extensive libraries
  • High demand in the job market with excellent career opportunities
  • Cross-platform compatibility (runs on Windows, Mac, Linux)

Python's philosophy emphasizes code readability with its notable use of significant whitespace. Its language constructs and object-oriented approach aim to help programmers write clear, logical code for small and large-scale projects.

Installing Python

Before you can start coding in Python, you need to install it on your computer. Follow these steps based on your operating system:

Windows Installation

  1. Visit python.org/downloads
  2. Download the latest Python installer for Windows
  3. Run the installer and check "Add Python to PATH"
  4. Click "Install Now" and wait for installation to complete
  5. Verify installation by opening Command Prompt and typing python --version

macOS Installation

  1. Visit python.org/downloads
  2. Download the latest Python installer for macOS
  3. Run the installer package and follow the instructions
  4. Verify installation by opening Terminal and typing python3 --version

Linux Installation

Most Linux distributions come with Python pre-installed. To verify, open Terminal and type:

python3 --version

If Python is not installed or you need a newer version, use your distribution's package manager:

Ubuntu/Debian:

sudo apt update
sudo apt install python3 python3-pip

Fedora:

sudo dnf install python3

Your First Python Program

Let's write the classic "Hello, World!" program to confirm everything is working:

hello_world.py 
print("Hello, World!")
# This is a comment - it won't be executed
print("Welcome to Python!")
Hello, World!
Welcome to Python!

Understanding Your First Program

  • print() is a built-in Python function that displays output to the screen
  • Text inside quotes "Hello, World!" is called a string
  • Lines starting with # are comments - Python ignores them
  • Python executes code line by line, from top to bottom

1. What does the following code display?

print("Python is " + "awesome!")
Python is+awesome!
Python is awesome!
Error: cannot add strings
"Python is " + "awesome!"
Correct! The + operator concatenates (joins) strings.
Incorrect. The + operator concatenates (joins) strings together.

2. Which of the following is a comment in Python?

// This is a comment
/* This is a comment */
# This is a comment
-- This is a comment
Correct! In Python, comments start with the # symbol.
Incorrect. In Python, comments start with the # symbol.

3. What command can you use to check your Python version in the terminal/command prompt?

python --check-version
python --version
python --ver
python --info
Correct! The --version flag displays your Python version.
Incorrect. Use python --version to check your installed Python version.
