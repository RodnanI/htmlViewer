This is a sample HTML file that will be displayed by your Next.js HTML file viewer.

How it works

When you add new HTML files to the /public/html-files directory, they will automatically appear on the homepage of your application.

You can then click on any of these files to view them using the built-in HTML viewer.

Try it yourself

Create your own HTML files and add them to the directory to see them appear on the homepage!
